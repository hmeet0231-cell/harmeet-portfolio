# Harmeet Singh - Portfolio

A stunning personal portfolio website for a creative developer and digital artist from Ludhiana, Punjab.

## 🌟 Features

- **Modern Design**: Glassmorphism UI with smooth animations
- **Color Theme**: Beautiful blue and green gradient accents
- **Responsive**: Fully mobile-responsive design
- **Sections**:
  - Hero section with animated text
  - About Me
  - Skills & Expertise (UI/UX Design, Python, Web Development, Drawing & Digital Art)
  - Featured Projects showcase
  - Contact with social links

## 🚀 Tech Stack

### Frontend
- React 19
- Tailwind CSS
- shadcn/ui components
- Lucide React icons
- React Router DOM

### Backend
- FastAPI
- MongoDB with Motor (async driver)
- Python 3.x

## 📦 Installation

### Frontend Setup
```bash
cd frontend
yarn install
yarn start
```

### Backend Setup
```bash
cd backend
pip install -r requirements.txt
uvicorn server:app --reload --host 0.0.0.0 --port 8001
```

## 🔧 Environment Variables

### Frontend (.env)
```
REACT_APP_BACKEND_URL=your_backend_url
```

### Backend (.env)
```
MONGO_URL=your_mongodb_connection_string
DB_NAME=your_database_name
```
